CONTROLS

  • JOYPAD CONTROL: (recommended!)
    • Analog stick --- Move your ship (digital pad might work as well)
    • A --- Fire!
    • B --- Dock on a stunned enemy vessel! (or fire missile)
      Insert the joypad before starting the game!

  • KEYBOARD CONTROL:
    • Arrow Keys --- Move your ship
    • X --- Fire!
    • C --- Dock on a stunned enemy vessel! (or fire missile)
    • F5 --- Pause
    • F12 --- Toggle scanlinemode
    • ESC --- Quit game

  • MOUSE CONTROL:
    • Go to the options menu and switch to mouse control!
    • Press the left mouse button and your ship moves towards the arrow.
    • Press the right mouse button only to cease fire. (afterburner)
    • Move the arrow into the docking marker to dock on a stunned enemy vessel.
    • Click at an enemy vessel to fire a homing missile. (if you have any)

  • TOUCH CONTROL:
    • Go to the options menu and switch to touch control!
    • Touch inside the control circle and your ship moves towards the arrow.
    • Move the arrow into the docking marker to dock on a stunned enemy vessel.
    • Touch an enemy vessel to fire a homing missile. (if you have any)
Cease fire to activate your afterburner for an extra speed boost!



A certain amount of enemies have to be destroyed to reveal the stargate to the next sector.



Ram and capture an enemy vessel to use enemy weapons...



Collect powerups to improve your interceptor...

Q and A

  • Q: Can I play in window mode?
    A: Yes, you can. Change the nanovoid.ini file in the game folder and add lines like this: 
    window
geometry=480x320
zoom=2
    This disables fullscreen mode and opens a window. (not recommended)
     
  • Q: Can I change the keyboard controls?
    A: Yes, you can. Change the nanovoid.ini file in the game folder and add a line like this: 
    key1=UDLRCA
    uses cursor keys (UDLR) for movement, CTRL to fire, ALT for docking 
    key1=wsadnm
    uses WASD keys for movement, N to fire, M for docking
    remark: all letters are case sensitive!
     
  • Q: Can I change the joypad controls?
    A: Yes, you can. Change the nanovoid.ini file in the game folder and add a line like this: 
    pad1=1122ab
    (the available axises are '1'-'8' and the buttons are 'a'-'z')
